# ZenYog - Design System

## Project Info
- **Title:** Design System Implementation
- **ID:** 8644327358966665826

## Brand Personality
"Transcendent Minimalist" — high-end, ethereal wellness experience combining modern science with ancient wisdom.

## Design Style
- **Minimalism** + **Glassmorphism**
- Extensive whitespace for calm atmosphere
- Light-refracting materials and hyper-thin structural lines

## Color Palette
| Color | Hex | Usage |
|-------|-----|-------|
| Surface | `#f3fbfb` | Primary canvas |
| Primary | `#006970` | Main actions |
| Accent (Aether Cyan) | `#00F0FF` | CTAs, active states |
| On Surface | `#151d1e` | Primary text |
| Secondary | `#5c5f60` | Secondary elements |

## Spacing
- **Unit:** 8px
- **Container Max:** 1280px
- **Gutter:** 24px
- **Section Padding:** 80px
- **Element Gap:** 16px

## Components
- **Primary Buttons:** Cyan (#00F0FF) background, white text, outer glow on hover
- **Ghost Buttons:** Transparent with hairline border, Cinzel Decorative text
- **Glass Cards:** White semi-transparent (70% opacity), 20px backdrop blur, 1px border
- **Inputs:** Minimalist bottom-border only, transitions to Cyan on focus
- **Chips/Tags:** Small uppercase Spectral labels, light gray background (#F4F4F5)

## Elevation
- **Frosted Glass:** `backdrop-filter: blur(20px)` with white/70% opacity
- **Hairline Borders:** 0.5px or 1px in very light gray (#E4E4E7)
- **Soft Shadows:** `0px 20px 40px rgba(0, 0, 0, 0.03)`

## Shape Language
- **Primary Radius:** 4px (Soft)
- **Roundness:** ROUND_FOUR
